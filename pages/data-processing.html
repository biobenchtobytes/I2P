Tidying

Tidying hones in on just the data you want to analyse.

The process includes sorting, filtering, and preprocessing raw data so that it can be easily analyzed and visualised.

Operating on groups

What if we want to perform arithmetic not on the whole dataframe, but on subsets? We can operate on chunks of a dataframe simultaneously using groupby.

loc_counts = df.groupby(['subcellular_location']).count()[['protein']]
loc_counts
protein
subcellular_location

What if we want to perform arithmetic not on the whole dataframe, but on subsets? We can operate on chunks of a dataframe simultaneously using groupby.

Think of this like creating filtered copies of the dataframe, conducting the operation, then combining them all back together again!
groupby is an iterator object which is not created until it is evaluated. Operations used with groupby create views and are not operating on the original dataframe.

We can access each chunk of the dataframe using a for loop:

for loc, data in df.groupby(['subcellular_location']):
  print(
    f'The average molecular weight of proteins in the {loc} is \
    {int(data["molecular_weight_kda"].mean())} kDa.')
